# jsgolf
Problems and solutions for the first Async.JS JavaScript Golf Night

# The Problems

Inputs and outputs were secret during the competition

```
// title  | Plus One!

// output | {"a": "b", "c": "d", "e": "f"}

// title  | Title Goes Here
```
